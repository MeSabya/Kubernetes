## Problem1
Given 
cat q2.json
```json
{
    "car": {
        "color": "blue",
        "price": "$20,000"
    },
    "bus": {
        "color": "white",
        "price": "$120,000"
    }
}
```
Extract bus element 

```shell
cat q2.json | jpath $.bus
```
## Problem2
Extract the price of the bus element.

```shell
cat q2.json | jpath $.bus.price
```
